Getting started
===========
`Bots` contains all accessible bots
```javascript
Bots["rover"]
```
This accesses the bot named "rover"
```javascript
Bots["rover"].move();
```
This "command" will move the bot
```javascript
Bots["rover"].turn_left();
Bots["rover"].turn_right();
```
These "commands" will turn the bot (left or right)
